# 📇 通讯录管理系统

一个基于 Web 的现代化通讯录管理系统，支持联系人的增删改查、收藏标记、多方式导入导出，并提供响应式界面。

## ✨ 功能特性

### 📌 核心功能
- **联系人管理**：添加、编辑、删除、查看联系人信息
- **收藏功能**：标记重要联系人，支持快速筛选查看
- **多种联系方式**：每个联系人可添加多个联系方式（手机、电话、邮箱、微信、QQ等）
- **智能搜索**：支持按姓名、公司、职位、备注、联系方式搜索
- **筛选功能**：按全部联系人或收藏联系人筛选显示

### 📊 数据管理
- **本地存储**：使用浏览器 localStorage 持久化数据
- **Excel 导入**：支持从 Excel 文件批量导入联系人
- **Excel 导出**：将全部联系人导出为格式化的 Excel 文件
- **数据统计**：实时显示联系人总数和收藏数量

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Edge 等）
- 网络连接（用于加载 Font Awesome 和 XLSX 库）

### 使用方法
1. **直接运行**：直接打开 `index.html` 文件即可使用
2. **添加联系人**：点击"添加联系人"按钮，填写信息并保存
3. **编辑/删除**：在联系人卡片上点击相应按钮操作
4. **收藏联系人**：点击联系人卡片上的星标图标
5. **导入导出**：使用顶部工具栏的导入/导出按钮

## 📱 响应式设计
- 支持桌面端、平板和手机访问
- 自适应布局，在不同设备上都能获得良好体验

## 🔧 技术栈
- HTML5 / CSS3 / JavaScript (ES6+)
- LocalStorage 本地存储
- XLSX.js (Excel 处理库)
- Font Awesome 图标库

## ⚠️ 注意事项
1. 数据存储在浏览器本地，清除浏览器数据会丢失所有联系人
2. 导入功能需要 Excel 文件格式符合模板要求
3. 建议定期导出备份重要联系人数据
